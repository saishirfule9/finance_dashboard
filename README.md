# Finance Dashboard UI

A simple and interactive finance dashboard built using React. This project demonstrates how to design a clean user interface, structure components, and manage data on the frontend.

dsa
---


## 🚀 Features

- Dashboard overview with Total Balance, Income, and Expenses  
- Transactions table with search functionality  
- Role based UI (Viewer and Admin)  
- Basic insights based on transaction data  
- Clean and responsive layout  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript  
- HTML5  
- CSS3

## 🎯 Functionality

- Users can view a financial summary including income, expenses, and balance  
- Transactions are displayed in a structured table format  
- Search functionality allows filtering transactions by category  
- Role switching:
  - Viewer: Can only view data  
  - Admin: Can add transactions  
- Insights section provides simple analysis of spending patterns

## ▶️ How to Run Locally

1. Clone the repository  
https://github.com/saishirfule9/finance_dashboard.git


2. Navigate to project folder  
cd finance-dashboard


3. Install dependencies  
npm install


4. Start the application  
npm start
